How to Convert PDF file to CSV file

• Preparation: the following software are required to conversion:
- Adobe Acrobat X
- Microsoft Excel
- Unikey
- Notepad++

• Step 1: Convert from PDF file to Excel file format
- Open PDF file with Adobe Acrobat

- choosing File → Save as → Spreadsheet → Microsoft Excel Workbookin order to convert PDF to Excel


- Name your file and click to Save

- Open your Excel file, we have this one


• Step 2: Encode the text to UTF-8 encoding and then save from Excel to CSV file
- Use key combination Ctrl + A to select everything and then copy to clipboard by Ctrl + C
- Right click to unikey and choose Toolkit or you can press Ctrl + Shift + F6


- In the Tookit, you set the source is VNI Windows and the destination is UTF-8 and tick to checkbox Do not use rich text. After finish setting, click to Convert button.


- Successfully converted message popup will be showed up, just click OK and you can close the Toolkit if you want.
- Come back to Excel, paste the clipboard back to the file by Ctrl + V, you will got this one


- The text will look bad, don’t worry, let it be.
- Now, convert the Excel file to CSV file by choosing File → Save as
- The Save As dialog appears, choose the Save as type is CSV (comma delimited) and then Save it.


- If there is any warning appears, just click Yes.
• Step 3: Now, you already have CSV file. Edit it with Notepad++


- You can see the text has become normally again in Notepad++ with UTF-8 encoding
- The work now is manually edit some redundant lines affect to the CSV format such as the page titles, headers , footers, etc… from Excel file.
You can use Replace All here quite useful.
- Finally, you got a CSV file converted from PDF file easily.
• NOTICE: with the big PDF file, it will take very long time to be able to convert from PDF to Excel file.
It’s should be split to small parts and convert one by one.

Searching Guide

There is 2 types of search that are simple search and advanced search.
Simple Search
User can enter any keyword in the text field. System will start searching when user stops typing or hit Search button.
QuickKar will display and bold relevant results that is related to the keyword.



Advanced Search
Advanced Search Button.



In the Advanced Search, there are 3 modes which are exact keyword, any keyword and none of the keyword.
Exact keyword means that the system will search and display the results that are exact from the user's input.
Any keyword means that the system will search and display the results that contains the keyword input.
None of the keyword menas that the system will search and display the results that do not contain the keyword input.
User can combine these function together in order to have more expected results.
After filling all fields, click Search button to start advanced search or click Cancel button to go back to the Simple Search.

Killer Feature 1: Let's Sing

When user double clicks on a song panel or click the micro button.




If user's network needs authentication to access the Internet, QuickKar will ask user for proxy setting. Click on proxy setting button to set proxy.
There are 4 fields need to be filled that are Host, Port, username and password.
Host: Enter proxy.rmit.edu.vn
Port: Enter 8080
Username: Enter your username
Password: Enter your password
Then Click Accept button to continue or click Cancel button to go back to User Frame.




When user clicks Accept button, a loading screen will display.
User just waits for few seconds so that system loads the song from the music website




If the selected song is unavailable from the music website, an error dialog is shown.




If the selected song is available and has more than 1 link, system will display available music link panel for user to choose.
Click into the link panel in order to listen to the selected song.




After clicking the link, QuickKar will open Window Media Player that contains the selected song.



The screen of the Media Player is in the Default Mode. If user wants to change the screen, click the button with 4 different colored dots



There are only 2 screens for user to choose.
User have to choose one screen.



To adjust the volume, hold the volume adjustment button and drag.
Move this button to left is turning the volume down and vice versa.



To mute the volume, click the speaker button.
If the Player is on Mute mode, click again to play sound.



Click the Play Button to play the song.

QuickKar is a software that helps users to search songs from the song list in karaoke club. Users do not have to find the song manually from the song book. As a result, the time on searching reduces so that the time for singing increases. This software also helps karaoke busineses to improve their service. When new album is released, business owners only need to download file from internet and import the file to the system. Moreover, the QuickKar has internal managing system so that business owners can assign their employees to manage the system. Employees can be assigned to be an administrator or a staff. The administrator has authority to manage the staffs' information. The administrator can add new account, edit or delete existed staff acounts. On the other hand, staff is able to work with song database. Staff has authority to add new song, edit pr delete existed songs.
